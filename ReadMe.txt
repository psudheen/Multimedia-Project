This simple java code reads in an rgb image of the specified format explained in the homework and then displays it on screen.
To run the code from command line, first compile with:

>> javac imageReader.java

and then, you can run for sample image as:

>> java imageReader sample_image.rgb width height

where, the first parameter is the image file name, second is the width and third is the height.

e.g) 
>> java imageReader image1.rgb 352 288
if the size of the image1.rgb is 352x288
